Ищите сотрудника для вашей компании?

Давайте вместе решим эту задачу

Шаг 1

Создайте карточку вакансии

При создании карточки вы можете:

  • Написать название и описание вакансии
  • Добавить дополнительных участников собеседования
  • Добавить поля для комментирования кандидатов

Шаг 2

Проводите собеседования

После создания карточки у вас есть возможность:

  • Переслать ссылку кандидатам для регистрации на собеседования
  • Скачать тестовое задание и резюме
  • Написать предварительно мнение о каждом кандидате

Шаг 3

Оценивайте кандидатов

После собеседования:

  • Оцените кандидатов в разделе вакансии
  • Добавьте дополнительный комментарий о соискателе
  • Наймите наиболее квалифицированного кандидата
